Web Platform for Energy Communities – Bachelor’s Thesis

Status: In Progress / Incomplete

Project Overview

This project is part of my Bachelor’s Thesis in Informatics and Telecommunications at the University of the Peloponnese.
It aims to develop a web-based platform where users can form and participate in an energy community, collaboratively produce renewable energy, and share the generated revenue among members.

The platform will be accessible through a web browser and will use modern web technologies, including Node.js for backend development.
